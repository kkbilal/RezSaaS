# Faz Bazlı Yol Haritası (Phase 0–5)

Bu yol haritası “kod yazmadan önce doğru parçalama” prensibiyle hazırlanmıştır. Her faz, **teslimat odaklı** ve test edilebilir çıktılara bağlanır.

## Fazlar

- `phase-0-kesif-karar.md`: Ürün tanımı + domain sözlüğü + risk kararları
- `phase-1-cekirdek-saas.md`: Çekirdek SaaS iskeleti (tenancy + RBAC + katalog + kaynak + temel booking)
- `phase-2-musteri-kesif-rezervasyon-mvp.md`: Keşif + işletme sayfası + slot seçimi + doğrulama + rezervasyon MVP
- `phase-3-guvenlik-operasyon.md`: Abuse, MFA, log hijyeni, backup/restore, operasyonel derinleşme
- `phase-4-odeme-gelir-optimizasyonu.md`: Depozito → prepayment → refund/webhook + analitik
- `phase-5-platformlastirma-genisleme.md`: API/webhook, çoklu şube gelişmiş, CRM/export, İYS, platformlaşma
